config
service in module appModernizationApp

Description

This module of the app allows you to configure the application with different parameters as well as define routes and associated templates.

The config function uses $routeProvider to define the routes for the application

Dependencies

  • $routeProvider

Methods

  • routeProvider('/', '/about', '/search', '/search/details', '/search/details/:param1', '/search/view/:param1/:param2', '/search/delete', '/search/delete/:param1')

    routeProvider

    This configuration is present in the JavaScript file app.js.

    This configuration is responsible for defining the routes of the application.

    Parameters
    ParamTypeDetails
    '/'object
    • views/main.html, MainCtrl
    '/about'object
    • views/about.html, AboutCtrl
    '/search'object
    • views/search.html, SearchCtrl
    '/search/details'object
    • views/details.html, DetailsCtrl
    '/search/details/:param1'object
    • views/details.html, DetailsCtrl - Edit Mode, param1: reservation id
    '/search/view/:param1/:param2'object
    • views/view.html, ViewCtrl - View Mode
    '/search/delete'object
    • views/delete.html, DeleteCtrl
    '/search/delete/:param1'object
    • views/delete.html, DeleteCtrl - Cancel Reservation
    Returns
    null

    Returns nothing.

    Example

    config(['$routeProvider', function($routeProvider, $httpProvider){});

uppercased
directive in module appModernizationApp

Description

This directive is used to convert whatever typed in the input type text or text area into an uppercase format.

Required ngModel. Does not have an associated template. Restricted: Element

Usage

as element:
<uppercased>
</uppercased>
as attribute
<input uppercased>
   ...
</input>
